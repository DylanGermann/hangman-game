# hangmane-game
 A simple hangmane game in Go
 
 ## How to run the program ?
 1. Make sure to install go 
 2. Go copy this project into your GOPATH
 3. Open the terminal in this folder
 4. For run 
 ```go
 go run ./main.go
 ```
 4bis. For build and create an .exe
 ```go
 go build ./main.go
```
